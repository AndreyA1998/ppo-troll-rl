# PPO vs TROLL Comparison on Atari Games

This project is devoted to the comparison of the training processes of a CNN agent to play the Atari games "Space Invaders" and "Breakout" via PPO and TROLL RL algorithms.

**Task.pdf** - problem statement,  
**PPO vs TROLL Comparison on Atari Games.pdf** - research report,  
**TROLL.pdf** - original article with [TROLL](https://openreview.net/forum?id=X9D5MVpPJ9) definition,  
**ppo_troll_rl.ipynb** - Jupyter notebook with experiment,  
**space_invaders.py** - utils for the "Space Invaders" environment,  
**breakout.py** - utils for the "Breakout" environment,  
**runner.py** - utils for rollouts sampling,  
**requirements.txt** - requirements for Python libs.

wandb.ai [report](https://api.wandb.ai/links/andreya1998-iai-msu/sk62c3mp) with interactive graphics.
